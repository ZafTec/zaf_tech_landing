Always use `bun` instead of `npm` or `npx` for installing packages, running scripts, and any CLI commands.

**Why:** User preference — they use bun as their package manager and runtime.
**How to apply:** Use `bun install`, `bun run`, `bunx` instead of `npm install`, `npm run`, `npx`.
